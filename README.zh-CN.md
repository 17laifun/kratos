## 🚀 使用说明

1. 下载最新发行版本的 .zip 格式安装包；

2. 登录 WordPress 管理员后台，进入 `外观` > `主题` 页面，然后点击 `添加` 按钮；

3. 进入添加主题页面之后，再点击 `上传主题` 按钮；

4. 点击 `选择文件`, 找到下载的 .zip 格式安装包，然后点击 `现在安装` 按钮；

5. 进入 `外观` > `主题` 页面，找到 Kratos 主题，然后点击 `启用` 即可。
